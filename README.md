# evcc-controller

## Setup S3 Mini Pro

### Enable flash mode

1. `Press + hold` the `0` button
2. `Click` the `Reset` button
3. `Release` the `0` button 

### Flash the firmware

```bash
cd config
esphome run lolin-s3-mini-pro.yaml
```

## Setup viewe C3 Rotary Encoder

### Firmware
